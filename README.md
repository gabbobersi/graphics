# graphics
C library to draw geometrich shapes in ppm files.
For prototyping purposes.
## Implemented shapes
- **graphic_fill**
  - Clear the image with background color.  
- **graphic_fill_rect**
  - Draw a rectangle.  
- **graphic_fill_circle**
  - Draw a circle.
- **graphic_save_to_ppm_file**: 
  - Create (overwrite) and save the ppm file.
